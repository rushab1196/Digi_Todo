# Digi Task Backend

This is a lightweight Laravel skeleton for the Digi Task application.

## Requirements
- PHP 8.3+
- Composer

## Setup
1. `composer install`
2. Copy `.env.example` to `.env` and configure your database
3. `php artisan migrate`
4. `php artisan serve`

## Testing
Run `composer test` to execute the PHPUnit test suite.
